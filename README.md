# MutiaraSurvey - Sistem Kuesioner Analisis Perilaku Komunikasi Pemain Roblox

MutiaraSurvey adalah platform kuesioner dan analisis berbasis web yang dirancang untuk mengidentifikasi kecenderungan pola perilaku komunikasi siswa (Sehat/Pasif, Berisiko, dan Toksik) pada platform *game online* Roblox. Sistem ini menerapkan teknik data mining dengan mengimplementasikan 4 algoritma komparatif: **K-Means, K-Medoids, DBSCAN, dan Hierarchical Clustering**, serta divalidasi menggunakan **Silhouette Score**.

## 🚀 Fitur Utama
* **Dashboard Analisis**: Menampilkan ringkasan visual total responden dan hasil akhir klastering.
* **Manajemen Data**: Pengelolaan data siswa, data kuesioner, dan rekaman jawaban responden.
* **Modul Clustering Komparatif**: Implemetasi langsung 4 algoritma clustering dalam satu sistem untuk perbandingan performa.
* **Uji Statistik & Validasi**: Integrasi metrik Silhouette Score untuk mengukur kualitas pengelompokan secara kuantitatif.
* **Ekspor Laporan**: Fitur ekspor data hasil pengelompokan ke format Excel.

## 🛠️ Teknologi yang Digunakan
* **Backend & Frontend**: PHP Native, JavaScript, Bootstrap 5
* **Database**: MySQL / MariaDB
* **Development Environment**: Laragon / XAMPP
